O que é HTML?

Hiper Text Markup Language (linguagem de marcação de texto) existe desde de 1991 e é a estrutura básica de uma página web. A estrutura do HTML é formada por um conjunto de elementos, as tags, elas são utilizadas para informar ao navegador que tipo de estrutura é essa que está sendo construída, podendo ser:

  • Títulos
  • Parágrafos
  • Imagens
  • Links
  • Etc

Descrições de algumas Tags importantes:

Antes de começarmos a falar das Tags, vamos aprender como usá-las. Elas sempre começam com o sinal de menor (<), e fecham com o sinal de maior (>), sempre acrescente uma barra (/) ao fechar as Tags, antes do sinal de maior (>).

Tag “!Doctype”

Voltar

Informa ao navegador que esse documento é do tipo html e sua versão.

Tag “html”

Voltar

Representa a raiz do documento, serve como um container que engloba todos os outros elementos html.

Tag “body”

Voltar

É onde fica todo o conteúdo de texto, imagens e vídeos em que o usuário vê e interage.

Tag “head”

Voltar

Contém as informações que serão lidas pelo navegador, que não têm interações com o usuário.

Tag “style”

Voltar

Usada para declarar estilos (CSS) para um documento.

Tag “p”

Voltar

Sinaliza parágrafo.

Tag “a href”

Voltar

Chamada de tag link, tem a capacidade de ir de um documento para outro documento.

Tag “h1 até h6”

Voltar

São tags de subtítulos e começam com “h1”, podendo ir até “h6”, à nível de importância. Importante elencar que só pode haver um h1 por página.

Tag “strong”

Voltar

É uma tag de formatação, e deixa o texto em negrito, dando ênfase ao conteúdo.

Tag “i”

Voltar

Também é uma tag de formatação, que deixa o texto em itálico.

Tag “u”

Voltar

Permite que o texto fique sublinhado, assim, formatando o texto.

Tag “hr”

Voltar

Acrescenta uma linha de divisão, deixando o documento com melhor aparência.

Tag “small”

Voltar

Faz com que o conteúdo fique pequeno, é uma tag à nível de formatação de texto.

Tag “del”

Voltar

Tem a função de deixar o conteúdo com a aparência de riscado (deletado).

Tag “sub”

Voltar

Permite que o texto inserido fique inferior. Ex.: An

Tag “sup”

Voltar

Deixa o texto inserido num grau de superioridade comparado a outro elemento. Ex.: ABbc

Tag “mark”

Voltar

Tem a função de deixar o documento com o aspecto de marca texto.

Tag “ol”

Voltar

Significa uma lista ordenada, necessita de uma ordem, e precisa de tags filhas (uma tag dentro da outra).

Tag “ul”

Voltar

Representa uma lista não ordenada, não necessita de uma ordem.

Tag “il”

Voltar

Sinaliza o item da lista, e é considerada uma tag filha.

Tag “abbr”

Voltar

É uma tag de citação, usada para abreviações.

Tag “blockquote”

Voltar

Também é uma tag de citações, porém citação de artigo.

Tag “font”

Voltar

É usada para alterar o tipo de fonte de um texto HTML.
